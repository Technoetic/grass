# 실시간 동기화 사용 가이드

## 문제 해결

### 실시간 동기화가 작동하지 않는 경우

1. **실시간 동기화가 실행 중인지 확인**
   ```bash
   # 새 터미널에서 실행
   python start_sync.py
   ```

2. **파일 변경 감지 확인**
   - 파일을 수정하고 저장하면 콘솔에 메시지가 나타나야 합니다:
     ```
     📝 [23:45:12] 파일 변경 감지: 251220_02.py
     ```

3. **디바운스 시간 확인**
   - 기본 5초 대기 후 커밋됩니다
   - 더 빠르게 하려면: `python sync_watcher.py --debounce 2`

## 사용 방법

### 기본 사용

```bash
# 터미널 1: 실시간 동기화 시작
python start_sync.py
```

### 파일 수정 테스트

1. 실시간 동기화 실행 중
2. Cursor에서 파일 수정 및 저장
3. 5초 후 자동 커밋 + 푸시

### 옵션

```bash
# 디바운스 시간 조정 (2초)
python sync_watcher.py --debounce 2

# 푸시 없이 커밋만
python sync_watcher.py --no-push
```

## 확인 방법

실시간 동기화가 작동하면:
- 파일 변경 시 콘솔에 메시지 표시
- 5초 후 자동 커밋 메시지 표시
- 자동 푸시 메시지 표시

---

**중요**: 실시간 동기화는 별도 터미널에서 실행해야 합니다!
