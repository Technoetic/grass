# 실시간 동기화 프로젝트

로컬 폴더와 Git 저장소를 실시간으로 동기화합니다.

## ⚠️ 중요: 실시간 동기화 사용법

### 1단계: 실시간 동기화 시작 (필수!)

**별도 터미널에서 실행해야 합니다:**

```bash
python start_sync.py
```

또는

```bash
python sync_watcher.py
```

### 2단계: 파일 수정

실시간 동기화가 실행 중이면:
- 파일을 수정하고 저장
- 콘솔에 "파일 변경 감지" 메시지 표시
- 5초 후 자동 커밋 + 푸시

## 확인 방법

실시간 동기화가 작동하면:
```
📝 [23:45:12] 파일 변경 감지: 251220_02.py
✅ [23:45:17] 커밋 완료: 자동 동기화: 251220_02.py (2025-12-20 23:45:17)
🚀 [23:45:18] 푸시 완료!
```

## 문제 해결

### 실시간 동기화가 작동하지 않는 경우

1. **실시간 동기화가 실행 중인지 확인**
   - 별도 터미널에서 `python start_sync.py` 실행
   - 실행 중이어야 파일 변경이 감지됩니다

2. **파일 변경 감지 확인**
   - 파일을 수정하고 저장하면 콘솔에 메시지가 나타나야 합니다

3. **디바운스 시간**
   - 기본 5초 대기 후 커밋됩니다
   - 더 빠르게: `python sync_watcher.py --debounce 2`

---

**중요**: 실시간 동기화는 별도 터미널에서 실행해야 합니다!
